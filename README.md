shortme
---------

### URL shortener service, powered by Go

[中文](./README_ZH.md)

## Features

- shortlen url
- redirect to url of hash

## RUN

```
git clone git@github.com:qichengzx/shortme.git
cd shortme
go run main.go
```

open "[http://localhost:8000](http://localhost:8000)"
